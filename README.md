# AstraGeek
This is a project for AstraGeek astronomy online school.
Here you can find some software for astronomical education.

## Release plan
0.1 Stereographic projection with planets, ecliptic, galactic equator, celestial equator.
_example_ https://skyatlas.app/star-charts/

0.2 Add precession of the Earth's rotation

0.3 Plotting particular parts of the sky. Pinhole projection.

0.4 Plotting constellations by the name (Enum class), fow and camera rotation angle

0.5 Animations of planets' transits (gif and static)

0.6 Create multiple examples of generated photos (Constellations folder and stereo projection)
