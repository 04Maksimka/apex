# 🌌 AstraGeek. Звездный атлас

Проект онлайн-школы астрономии AstraGeek.
Здесь вы можете найти программное обеспечение для создания звездных карт.

На [сайте](https://skychart.astrageek.ru/) проекта AstraGeek вы можете создавать классические небесные диаграммы и карты звездного неба.

Также посетите сайт [astrageek.ru](https://astrageek.ru/), где вы найдете обучающие видео по астрономии и на смежные темы.

---

## Особенности

1. Классическая стереографическая проекция всего неба, доступного к наблюдению в указанном месте и в указанное время.
2. Режим камеры-обскуры для определенной небольшой области неба в указанном направлении (выбранное созвездие)
3. Можно добавлять различные дополнительные опорные элементы: координатные сетки, большие координатные круги (эклиптика, галактический и небесный экватор), стороны света, точку зенита и небесные полюса.
4. Можно добавить к изображению контуры всех видимых созвездий или отдельных выбранных созвездий, а также названия каждого видимого или избранного созвездия.

## Как использовать

**Если Вы хотите генерировать карты самостоятельно, то переходите сразу к 
разделу документации ниже**

Вы можете пользоваться готовыми наборами карт звездного неба, которые мы уже сгенерировали. Для этого:

1. Перейдите по [ссылке](https://drive.google.com/drive/folders/1AfBc7pkqLH_65TMAeXvpzzzqrLcVKNFO?usp=sharing)
2. Скачайте архивы с картами режима камеры-обскуры и скайчартами.

Внутри каждой папки находятся сгруппированные по номерам карты звездного неба с заданиями. 

Внутри папки **pinhole_samples** находятся две папки:

1. **constellations**  - карты с конкретным центральным созвездием, для каждого из 88 созвездий
2. **random_sky** - карты с произвольным центральным созвездием, 10 примеров

Внутри папки **stereographic_samples** сгруппированы скайчарты на произвольные дату и место

Каждая группа состоит из трех файлов карт:

1. Для ученика без планет, пустой классический скайчарт - _student.pdf_
2. Для ученика с планетами - _student_with_planets.pdf_
3. Для учителя - _teacher.pdf_

Каждый файл состоит из двух страниц: на первой находятся задания к карте, на второй непосредственно сама карта.

Файл для ученика содержит только задания и "голую" карту. Файл для учителя содержит все ответы и заполненную карту
