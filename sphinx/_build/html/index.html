AstraGeek Sky Chart Tools¶

Software for building star maps and sky charts, developed for the AstraGeek astronomy online school.

  • Build classic sky charts and star maps via the web interface

  • Use the astrageek CLI to generate maps locally from the command line

  • Educational astronomy videos: astrageek.ru
