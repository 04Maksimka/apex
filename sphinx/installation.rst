Installation
============

.. note::
   It is recommended to install AstraGeek inside a virtual environment
   to avoid dependency conflicts.

Setting up a virtual environment
---------------------------------

**Linux / macOS**

.. code-block:: bash

   python3 -m venv .venv
   source .venv/bin/activate

**Windows**

.. code-block:: bat

   python -m venv .venv
   .venv\Scripts\activate

From PyPI
---------

.. code-block:: bash

   pip install astrageek

From source (GitHub)
--------------------

.. code-block:: bash

   git clone https://github.com/04Maksimka/AstraGeek.git
   cd AstraGeek
   pip install -e .
