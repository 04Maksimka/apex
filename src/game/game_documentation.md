## Быстрый старт

### 1. Регистрация Blueprint в `app.py`

Добавьте следующие строки в `src/web/app.py`:

```python
from src.web.game_blueprint import game_bp
app.register_blueprint(game_bp)
```

### 2. Запуск

```bash
python -m flask --app src/web/app run
# или
astrageek serve
```

### 3. Открыть в браузере

```
http://localhost:5000/game/lobby
```

Вы увидите лобби с пятью режимами игры, выбором сложности и количеством раундов.

## Архитектура

```
src/
├── game/
│   ├── __init__.py          # Экспорты модуля
│   ├── session.py           # Управление игровыми сессиями (in-memory)
│   ├── question_factory.py  # Генерация вопросов для каждого режима
│   └── scoring.py           # Подсчёт очков, ранги, бонусы
│
└── web/
    ├── game_blueprint.py    # Flask Blueprint: /game/*
    └── public_html/
        ├── games.html           # Лобби — выбор режима и сложности
        ├── game_constellation.html  # Режим 1: Угадай созвездие
        ├── game_star.html           # Режим 2: Назови звезду
        ├── game_messier.html        # Режим 3: Объекты Мессье
        ├── game_draw.html           # Режим 4: Нарисуй созвездие
        └── game_trivia.html         # Режим 5: Trivia + Карта
```

### Поток данных

```
Браузер  →  GET /game/lobby         → games.html
Браузер  →  POST /game/api/start    → создаёт GameSession
Браузер  →  GET  /game/api/question → QuestionFactory генерирует изображение
                                      (Pinhole/Stereo проекция через matplotlib)
                                      → base64 PNG + опции
Браузер  →  POST /game/api/answer   → scoring.calculate_score → результат
Браузер  →  GET  /game/api/hint     → подсказка (-50% очков)
```

## Игровые режимы

### Режим 1 — Угадай созвездие 🔭

**Цель:** Определить созвездие по снимку области неба без подписей.

**Как работает:**
- Сервер генерирует изображение через `PinholeProjector`, центрированное на случайном созвездии
- Линии созвездия отображаются (`add_constellations=True`), подписи скрыты (`add_constellations_names=False`)
- Игроку предлагается 4 варианта: 1 правильный + 3 дистрактора из того же уровня сложности

**Сложность:**
| Уровень | Количество созвездий | Лим. блеска |
|---------|---------------------|-------------|
| Легко   | 15 знаменитых       | 4.5m        |
| Средне  | ~37 созвездий       | 5.5m        |
| Сложно  | Все 88              | 6.5m        |

**Очки:** 100 / 200 / 350 базовых × бонусы

---

### Режим 2 — Назови звезду ⭐

**Цель:** Определить именную звезду, выделенную красным кольцом.

**Как работает:**
- Выбирается случайная именная звезда из каталога `lines_data.json` (HIP ID → название)
- Генерируется узкоугловое изображение (FOV = 30°) с камерой, направленной прямо на эту звезду
- Звезда подсвечивается двойным красным кольцом в центре изображения
- 4 варианта: настоящее имя + 3 других именных звезды

**Источник данных:** `src/constellations_metadata/lines_data.json`  
Пример: `"HIP 32349": [{"native": "Sirius", "english": "Sirius"}]`

**Сложность:**
| Уровень | Пул звёзд               |
|---------|------------------------|
| Легко   | Ярчайшие (≤ 4.5m)      |
| Средне  | Средние (≤ 5.5m)       |
| Сложно  | Все именные (≤ 6.5m)  |

---

### Режим 3 — Объекты Мессье 🌌

**Цель:** Определить номер объекта Мессье по изображению области неба.

**Как работает:**
- Выбирается случайный объект из `MessierCatalog` (110 объектов)
- Изображение центрируется на координатах объекта (FOV = 50°)
- Вопрос содержит тип объекта и созвездие (без номера)
- 4 варианта: M1–M110 (правильный + 3 случайных)

**Типы объектов Мессье в игре:**
- Галактика (GALAXY)
- Шаровое скопление (GLOBULAR_CLUSTER)
- Рассеянное скопление (OPEN_CLUSTER)
- Туманность (NEBULA)
- Остаток сверхновой (SUPERNOVA_REMNANT)
- Звёздное облако (STAR_CLOUD)

---

### Режим 4 — Нарисуй созвездие ✏️

**Цель:** Соединить звёзды созвездия линиями, как на официальных картах.

**Как работает:**
1. Сервер возвращает список звёзд с нормализованными координатами `(x, y) ∈ [-1, 1]`
2. Координаты вычисляются через гномоническую проекцию на касательную плоскость в центре созвездия
3. Игрок рисует на HTML Canvas: кликает одну звезду → другую → появляется линия
4. После нажатия «Проверить» сравниваются нарисованные рёбра с эталонными

**Оценка:**
```
совпавшие_рёбра / всего_рёбер × 100%
≥ 50% → зачтено как правильный ответ
```

**Управление (canvas):**
- `Клик` по звезде → выбрать / снять выбор
- `Клик` по второй звезде → нарисовать линию
- `↩ Отменить` → убрать последнюю линию
- `🗑 Очистить` → стереть все линии
- `✔ Проверить` → отправить ответ

**После проверки** на Canvas зелёными пунктирными линиями показывается эталонное созвездие.

---

### Режим 5 — Trivia + Карта 🗺️

**Цель:** По текстовому описанию созвездия выбрать его правильное название.

**Как работает:**
- Сервер выдаёт вопрос-описание (мифология, яркие звёзды, особенности)
- Генерируется широкоугольная карта (FOV = 120°) с подписями созвездий
- Игрок читает описание, смотрит на карту и выбирает один из 4 вариантов

**База вопросов (`TRIVIA_QUESTIONS`):**
- 10+ вопросов покрывают главные созвездия (Орион, Большая Медведица, Кассиопея и др.)
- Каждый вопрос содержит уникальную астрономическую деталь

**Расширение базы:**  
Добавить вопросы в `src/game/question_factory.py` в список `TRIVIA_QUESTIONS`:
```python
{
    "question": "...",
    "answer": "ORI",   # аббревиатура 3 буквы из CONSTELLATIONS_DATA
    "hint": "...",
}
```

## API Reference

Все эндпоинты доступны по префиксу `/game/api/`.

### `POST /game/api/start`

Создаёт новую игровую сессию.

**Тело запроса:**
```json
{
  "mode": "constellation",
  "difficulty": "easy",
  "rounds": 10
}
```

**Допустимые значения:**
- `mode`: `constellation` | `star` | `messier` | `draw` | `trivia`
- `difficulty`: `easy` | `medium` | `hard`
- `rounds`: 1–30 (по умолчанию 10)

**Ответ `201`:**
```json
{
  "session_id": "uuid-...",
  "mode": "constellation",
  "difficulty": "easy",
  "score": 0,
  "round": 0,
  "total_rounds": 10
}
```

---

### `GET /game/api/question?session_id=<id>`

Возвращает следующий вопрос.

**Ответ для режимов с изображением (`constellation`, `star`, `messier`, `trivia`):**
```json
{
  "session": { ... },
  "question_type": "constellation",
  "question": "Какое созвездие изображено на снимке?",
  "image": "<base64 PNG>",
  "options": ["Orion", "Cassiopeia", "Leo", "Cygnus"],
  "hint": "...",
  "round": 1
}
```

**Ответ для режима `draw`:**
```json
{
  "session": { ... },
  "question_type": "draw",
  "question": "Нарисуйте созвездие...",
  "stars": [
    { "hip_id": 27989, "x": 0.12, "y": -0.34, "v_mag": 0.45, "name": "Betelgeuse" },
    ...
  ],
  "hint": "...",
  "round": 1
}
```

**Ответ, если игра завершена:**
```json
{
  "finished": true,
  "score": 1850,
  "rank": "⭐ Star Navigator",
  "correct_count": 8,
  "accuracy": 80.0
}
```

---

### `POST /game/api/answer`

Отправляет ответ и получает результат.

**Тело для режимов с выбором:**
```json
{
  "session_id": "uuid-...",
  "answer": "Orion",
  "used_hint": false,
  "time_seconds": 6.3
}
```

**Тело для режима `draw`:**
```json
{
  "session_id": "uuid-...",
  "drawn_edges": [[27989, 24436], [24436, 26727]],
  "used_hint": false,
  "time_seconds": 42.1
}
```

**Ответ:**
```json
{
  "correct": true,
  "correct_answer": "Orion",
  "player_answer": "Orion",
  "points_earned": 150,
  "total_score": 350,
  "streak": 2,
  "best_streak": 2,
  "round": 3,
  "total_rounds": 10,
  "accuracy": 100.0,
  "fun_fact": "Орион содержит две из самых ярких звёзд...",
  "is_finished": false,
  "rank": "🌑 Beginner"
}
```

---
